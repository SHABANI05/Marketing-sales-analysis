# Marketing-sales-analysis
python analysis

## Project Overview
This project analyzes  sales data-set to understand store performance,
department performance, and holiday sales patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Sales by store
- Sales by department
- Holiday vs non-holiday sales
- Monthly sales trends

## Key Insights
- Some stores generate significantly higher sales.
- Certain departments contribute most of the revenue.
- Holiday weeks show higher average sales.
- Sales vary across months indicating seasonal demand.
